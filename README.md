# LXND × A$AP Rocky Tour — Stage Visuals Case Study

A public case study documenting how Cravin' Adventure Studios produced a hyper-real war-scene stage loop for **LXND**'s full set on A$AP Rocky's *Don't Be Dumb* tour, brainstormed, built, and delivered in a single same-night sprint (about a 3.5-hour turnaround, June 25, 2026).

**Live page:** https://cravinadventure.github.io/lxnd-asap-case-study/

## What's inside

The story, end to end:

- **How it started** — a phone call and the DMs that opened the door
- **The war room** — the group-chat brainstorm and Mithran's mood board
- **Locking the brief** — the questions and the proposal PDF
- **The build** — concept to shot list, the full clip set
- **Red to green** — the one-move color flip and the After Effects comp breakdown
- **The QR code** — baked into the world on a soldier's back vest, timed to surface about once a minute across the 10-minute run
- **The handoff** — the clean, labeled client file system and Dropbox delivery

## Tech

Single-file static site: one `index.html` with inline CSS and inline SVG, plus an `assets/` folder of images and video. No build step, no dependencies. Served by GitHub Pages.

Run locally:

```bash
python3 -m http.server 4187
# open http://localhost:4187
```
